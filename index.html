安裝方法

尚未上架 Chrome Web Store，請依以下步驟手動安裝，約 3 分鐘完成。

Installation

Not yet on the Chrome Web Store. Install manually in about 3 minutes.

1

下載並解壓縮Download & Unzip

點上方按鈕下載，解壓縮後會得到一個 breath-moment 資料夾。

Click the download button above and unzip it. You'll get a folder called breath-moment.

2

開啟擴充功能頁面Open Chrome Extensions

在 Chrome 網址列輸入並按 Enter：

In your Chrome address bar, type and press Enter:

chrome://extensions
3

開啟開發者模式Enable Developer Mode

頁面右上角有「Developer mode」開關，將它打開。

Toggle on Developer mode in the top-right corner of the page.

4

載入擴充功能Load the Extension

點左上角「Load unpacked」，選擇剛才的 breath-moment 資料夾。

Click Load unpacked in the top-left, then select the breath-moment folder.

✓ 開啟新分頁，Breath Moment 就在那裡了。 Open a new tab — Breath Moment will be there.
想移除？To uninstall: 回到 chrome://extensions，找到 Breath Moment，點「Remove」即可。 Go back to chrome://extensions, find Breath Moment, and click Remove.
