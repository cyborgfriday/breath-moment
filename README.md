# Breath Moment

---

## 設計理念

*Breath Moment* 在你忙碌的時刻，靜靜支持你，
在每一個新分頁裡，為你留一朵雲的空間。

不是要你停下來，
而是在你前進的途中，
讓呼吸帶你回到自己——
一次吸氣，一次屏息，一次呼氣。

這一分鐘，是你給自己的。

---

## Design Philosophy

*Breath Moment* quietly supports you through the busyness —
holding a cloud of stillness in every new tab, just for you.

Not asking you to stop,
but offering a moment to return to yourself —
one inhale, one hold, one exhale.

This minute is yours.

---

## 功能

- **4-7-8 呼吸法** — 緩解焦慮最適合的呼吸法。吸氣 4 秒，屏息 7 秒，呼氣 8 秒，三個循環，約一分鐘
- **每日紀錄** — 記錄你今天練習了幾次、累積了多少時間
- **三種主題** — Riverside（河岸晨光）、Lavender Dawn（薰衣草黎明）、Nocturne（夜曲）
- **五種語言** — 英文、繁體中文、日文、韓文、西班牙文
- **無帳號，無追蹤** — 所有資料僅存於本機瀏覽器

---

## Features

- **4-7-8 Breathing** — The most effective breathing technique for relieving anxiety. Inhale 4s · Hold 7s · Exhale 8s · Three cycles · About one minute
- **Daily log** — Tracks how many sessions and how many minutes you've breathed today
- **Three themes** — Riverside, Lavender Dawn, Nocturne
- **Five languages** — English, 繁體中文, 日本語, 한국어, Español
- **No account, no tracking** — All data stays in your local browser

## 安裝方法（測試版）

> 目前為開發者測試版，尚未上架 Chrome Web Store。
> 請依以下步驟手動安裝：

### 步驟一：下載擴充功能資料夾

從朋友提供的連結下載 `breath-moment.zip`，解壓縮後會得到一個 `breath-moment` 資料夾。

### 步驟二：開啟 Chrome 擴充功能管理頁面

在 Chrome 網址列輸入：

```
chrome://extensions
```

### 步驟三：開啟開發者模式

點擊右上角的「**Developer mode**」（開發人員模式）開關，將它打開。

### 步驟四：載入擴充功能

點擊左上角「**Load unpacked**」（載入未封裝項目），
選擇剛才解壓縮的 **`breath-moment`** 資料夾。

### 完成 ✓

開啟新分頁，*Breath Moment* 就會出現。

## Installation (Beta)

> This is a developer preview, not yet on the Chrome Web Store.
> Please follow these steps to install manually:

### Step 1 — Download the extension folder

Download `breath-moment.zip` from the link provided, and unzip it. You'll get a folder called `breath-moment`.

### Step 2 — Open Chrome Extensions

In your Chrome address bar, type:

```
chrome://extensions
```

### Step 3 — Enable Developer Mode

Toggle on **Developer mode** in the top-right corner of the page.

### Step 4 — Load the extension

Click **Load unpacked** in the top-left corner,
then select the **`breath-moment`** folder you unzipped.

### Done ✓

Open a new tab — *Breath Moment* will be there.

---

### 想移除怎麼辦？ / How to uninstall?

回到 `chrome://extensions`，找到 *Breath Moment*，點擊「**Remove**」即可。

Go back to `chrome://extensions`, find *Breath Moment*, and click **Remove**.
